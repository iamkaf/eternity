Why Eternity?

The old way

  • Proprietary software, expensive licenses
  • Can't collaborate — files don't work with Git
  • Locked into one export format
  • No multiplayer support
  • Hard to extend with new features

The Eternity way

  • Free and open-source, forever
  • Git-friendly — each character, item, map is its own file
  • Export to desktop, web, and mobile
  • Built-in multiplayer
  • Plugin system for limitless extensions

Tell Your Story

Two ways to create game logic — no programming experience needed.

Beginner friendly

Event Commands

Point-and-click storytelling. Show dialogue, give items, move characters, trigger battles — all from a simple command list. Just like RPG Maker.

Advanced

Visual Scripting

Drag-and-drop node graph for complex logic. Connect condition, action, and flow nodes visually. See your game logic as a diagram.

For programmers: TypeScript scripting sandbox with full engine API access

Epic Battles

Classic turn-based combat with modern flexibility.

  • ✓ Turn-based party vs. enemy troop battles
  • ✓ Attack, Defend, Skills, Items, Escape
  • ✓ Custom damage formulas per skill
  • ✓ Status effects with durations and conditions
  • ✓ Enemy AI with weighted action patterns
  • ✓ Battle animations and visual effects
  • ✓ Active Time Battle (ATB) as alternative mode
  • ✓ Fully extensible via plugins
